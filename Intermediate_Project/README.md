# Titanic Survival Prediction using KNN

## Description
This project predicts passenger survival using the K-Nearest Neighbors (KNN) algorithm with advanced preprocessing and parameter tuning.

## Dataset
- Titanic dataset (train_KNN.csv)
- Contains passenger details such as age, sex, class, fare, etc.

## Steps Performed
1. Data Cleaning (handled missing values)
2. Feature Engineering (encoding categorical variables)
3. Data Scaling (StandardScaler)
4. Model Building (KNN)
5. Hyperparameter Tuning (best K value)
6. Evaluation (accuracy, confusion matrix, classification report)

## Results
- Best K value selected using accuracy graph
- Model achieved good prediction accuracy
- Gender and class significantly influence survival

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Conclusion
KNN performs well for classification when properly scaled and tuned. Selecting optimal K improves accuracy significantly.
